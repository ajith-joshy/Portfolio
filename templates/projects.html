TweetPulse(2024-25)

Designed and implemented a real-time sentiment analysis system for X(Twitter), focusing on specific hashtags. The system includes key features such as an alert mechanism for detecting sudden sentiment shifts, trend detection to analyze emerging topics, and an interactive dashboard to visualize sentiment trends and data insights. This solution enables effective monitoring of public opinion, offering actionable intelligence for decision-making and engagement strategies.

I selected Apple as the brand for monitoring, where tweets containing the #apple keyword were collected and analyzed for sentiment. The tweet collection process began in October 2024 and continued until February 2025, during which approximately 30,000 tweets were gathered using Twikit (a Python library). From these, about 26,000 tweets were used for model selection on multiple machine learning models.

Among all the algorithms tested, XGBoost provided the highest accuracy based on various evaluation metrics. The best-performing model, XGBoost, was then deployed for continuous sentiment prediction on newly collected tweets. Finally, the results were visualized through an interactive and dynamic Power BI dashboard, enabling clear interpretation of sentiment trends over time.
